'list.info' contains the list of all sources that will be used by the program.

WARNING - All sources in 'list.info' must start with 'sources/' followed by the name of the source file

EXAMPLE - 'sources/test1.txt' for file 'test1.txt' in folder 'sources'

Source files can be added in 'list.info' by using the program in menu 'Manage sources'
